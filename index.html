I really like playing video games, so let's talk about my favourites!

First up, Geometry Dash.

Geometry Dash is a rhythm-based platformer game where players control a cube-shaped icon as it navigates through levels filled with obstacles and music.

The game emphasizes timing and reflexes, as players must jump, fly, and dodge obstacles to the beat of the music. It features a level editor for creating custom levels and playing those created by others.

Now onto Roblox!

Roblox is an online platform where users can play and create games, essentially functioning as a massive multiplayer online game (MMO) and game creation system.

It allows anyone to design and share their own "experiences" (games) using the platform's tools and play games created by others.

Roblox also incorporates social features, allowing users to interact and create together.

Roblox is free and available on all platforms! So all you have to do to get it is search Roblox on pc, mobile or even console and you will be able to download it.
